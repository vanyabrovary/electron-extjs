# electron-extjs

### Описание

Каркас кроссплатформенного десктоп-приложения: UI на ExtJS, обёртка - Electron. 
Запускается на Windows, macOS, Linux.

### Основная идея

Возможность создавать JS приложения с UI на ExtJS, которые будут запускаться на MacOS, Linux, Windows.

### Отличие Electron от типичных веб-приложений

В типичных веб-приложениях UI общается с серверной частью по TCP: запросы уходят на сокет веб-сервера, который их обрабатывает. 

В Electron веб-сервер для этого не применяется (мы не можем запустить нашь бекенда на TCP сокете).
Роль серверной части выполняет main-процесс, транспортом служит встроенный IPC. 

Данный каркас воспроизводит поверх IPC ту же MVC-схему. 
ExtJS может обращается к main-процессу, как к обычному Store.

### AppImage
